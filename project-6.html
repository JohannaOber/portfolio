HMI for Driving Simulation

A modular, brand-aligned EV cockpit interface built for Akkodis's internal driving simulation software, ready for animation and system integration.

Client

Akkodis

Internal Project · Solo

My Tasks

ConceptUI DesignComponent Library

Challenge

A generic HMI that needs to feel real

Akkodis needed a clean, generic electric vehicle HMI to showcase the capabilities of their driving simulation software to potential clients. The interface had to reflect the Akkodis corporate identity, cover all essential EV functions, and be fully modular, ready for animation, system signal integration, and 3D vehicle extensions.

01 Discovery

Benchmarking EV Interfaces

I reviewed existing EV cockpit interfaces and HMI design principles to identify the essential elements for a convincing simulation environment. The goal was a generic interface, not tied to any specific vehicle brand, that still felt technically credible.

02 Define

Scope and Component Planning

I defined the full set of HMI elements: speed, range, drive modes, turn signals, system states, and media controls. Each element was planned as a standalone component to enable flexible animation and system integration in later phases.

03 Design

Reduced, Functional UI

The interface uses a deliberately minimal visual language, with clean typography, structured layout, and the Akkodis brand palette. Every element is designed to communicate clearly at a glance, even in a simulated driving context.

04 Test

Component Handoff and Review

The completed component library was reviewed against the simulation team's technical requirements and adjusted for animation readiness. All elements were annotated and structured for efficient handoff to the engineering team.

Solution & Impact

Modular, brand-ready, simulation-tested

The result is a complete, modular EV HMI covering all core vehicle functions in a clean, Akkodis-branded interface. Every UI element is provided as an individual component in a structured library, ready for animation, signal input, and 3D integration. The design is intentionally generic, ensuring it remains neutral across different automotive contexts.

The HMI component library streamlined the simulation team's animation workflow and provided a solid foundation for client demonstrations. The project serves as a base product to clearly and realistically communicate the possibilities of working with Akkodis.
